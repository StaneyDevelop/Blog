В данном курсе мы будем писать клиент-серверное приложение для сайтов, работающих на CMS WordPress. Курс разбит на несколько частей, и эта будет посвящена основам и созданию списка. Здесь будут затронуты темы планирования архитектуры, создания логики и UI.

Необходимый уровень знаний
Данные уроки будут интересны и полезны для начинающих, так как будет сделан упор на элементарные области связанные с программированием на Android. Применяя эти знания, можно будет быстрее начать использовать более сложные и новые инструменты. Стоит уточнить, что для усвоения данного материала необходимо, чтобы читатель уже обладал знаниями основ JavaSE (классы, интерфейсы, пакеты), навыками работы с Android Studio и Gradle (сборка проекта, настройка устройства для запуска и отладки приложений), знаниями иерархии файлов проекта на Android (исходный код, ресурсы)
Теперь нужно определиться с задачей и обдумать варианты реализации решения. Задача подразумевает получение, хранение, обновление и вывод информации о статьях с сайта. Разделим это на части. Предположим, что у нас будет начальный экран, который будет демонстрировать превью статей.
Для решения мы создадим контейнер, который будет в себе содержать список, каждый элемент которого отображает данные о конкретной статье.
Создайте новый проект в Android Studio. Приложение предназначено только для смартфонов, поэтому выбирайте целевой девайс только Phone and Tablet. В шаблонах выберите Add No Activity. В данных статьях мы будем поддерживать минимальную версию SDK 15. Убедитесь в том что gradle подтянул зависимость com.android.support:appcompat-v7. Это необходимо для использования в проекте особенных возможностей некоторых классов и темы материального дизайна на версиях SDK меньше 21.
